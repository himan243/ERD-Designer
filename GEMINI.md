# Project Context: ERD (Entity Relationship Diagram)

This project consists of a full-stack application with a React-based client and a Node.js-based server.

## Tech Stack

### Client
- **Framework:** React 18+ (TypeScript)
- **Build Tool:** Vite
- **Styling:** Vanilla CSS (styles.css)
- **Linting:** ESLint

### Server
- **Runtime:** Node.js
- **Entry Point:** index.js

## Project Structure
- `client/`: React frontend application.
- `server/`: Node.js backend application.

## Development Guidelines
- Always maintain type safety in the client using TypeScript.
- Prefer vanilla CSS for styling as per general mandates.
- Keep client and server logic decoupled.
